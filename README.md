A comprehensive financial model for a 3 lakh sq ft residential real estate development project. The model includes phased construction (20-30-30-20), detailed cost breakdowns (construction, marketing, government approvals), capital structure (equity and debt), monthly sales projections, and scenario analysis for pricing and marketing budgets. NPV, IRR, and sensitivity analysis are calculated using Excel. Ideal for learning real estate finance, project modeling, and scenario-based decision-making.
